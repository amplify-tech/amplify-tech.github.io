Platform for programming

Where to code ???
There are IDE available for programming. IDE is a software ,an application , a platform for programming purposes, where you can write a code and run the code. It contains editor , compiler , debugger , library ... .
Now at the starting level you can say code is a task that you want to get performed by computer, written in a computer language.
Now editor is a platform where you will write the code.
And the compiler translates the code into a readable language that the computer is going to execute . Debugger check whether your code is correct or not . If there is some mistakes it will show the error and the point of error , it can give suggestions. If the code is correct then program will give correct output on giving required input.
