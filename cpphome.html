conversion

  1. Implicit (automatic type conversion)

    2. Done by the compiler on its own. type promotion takes place.
    bool ⇨ char ⇨ short int ⇨ int ⇨ unsigned int ⇨ long ⇨ unsigned ⇨ long long ⇨ float ⇨ double ⇨ long double

    int x = 10 + 'a'

    double z = x + 2.5

  2. Explicit (type casting)
    • Converting by assignment:

      • (type) expression

      double x = 1.2;

      int sum = (int)x + 0.8; //output sum=1

    • using Cast operator:

      • float f = 3.5;

      int b = static_cast(f);

With function overloading, multiple functions can have the same name with different parameters (only datatype can be changed).
