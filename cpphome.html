Intro

filename:- filename.cpp
compile :- g++ filename.cpp
Run program :- ./a.out
first save file as filename.cpp then compile then run. A file named a.out will automatically create during compilation of cpp program.

Variables

Variables are used to store different values and dealing with them. Datatypes are the types of variables. Some of the datatypes in cpp are int, float, double, char, string, bool, void ...

In c++ cin is used to take input something.
